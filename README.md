# ECON 151 Study Tool

Interactive study tool for **UCSB ECON 151: Economics of Gender** (Prof. Lundberg, Winter 2026).

Built with React + Vite. Covers all 10 weeks of course material.

## What's inside

| Mode | Description |
|------|-------------|
| 📖 **Learn** | Accordion-style content browser — every paper, every concept, organized by theme |
| 🎴 **Flashcards** | 38 cards covering all 8 themes. Tap to flip, track mastery |
| 🎯 **Quiz** | 20 multiple-choice questions, 10 per round. Instant feedback + missed-question review |
| 🧩 **Matchup** | Match papers to their findings. 16 papers in the bank, 6 per round |
| ⚡ **Number Blitz** | 60-second timed challenge — type key stats as fast as you can (24 stats) |
| 🔗 **Connection Challenge** | Two random papers appear. Think about how they relate, then reveal the connection (8 pairs) |
| ⚖️ **T/F/U Showdown** | Lundberg-style True/False/Uncertain statements with model explanations (7 statements) |
| ⏱️ **Pomodoro** | 25/5 focus timer with session tracking |

## Quick start

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/econ151-study-tool.git
cd econ151-study-tool

# Install dependencies
npm install

# Run locally
npm run dev
```

Then open http://localhost:5173 in your browser.

## Deploy to GitHub Pages (free hosting)

1. Push to GitHub
2. Install gh-pages: `npm install --save-dev gh-pages`
3. Add to package.json scripts: `"deploy": "vite build && gh-pages -d dist"`
4. Run: `npm run deploy`
5. Go to repo Settings → Pages → Source: gh-pages branch
6. Your tool is live at `https://YOUR_USERNAME.github.io/econ151-study-tool/`

## Deploy to Vercel (even easier)

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com), sign in with GitHub
3. Import your repo
4. Click Deploy — that's it

## Topics covered

1. Changing Economic Roles of Men and Women (Goldin 2006, Coile & Duggan 2019)
2. Educational Gender Gaps (GKK 2006, Brenoe & Zölitz 2020, Lundberg 2020)
3. Family Economics & Bargaining (Becker, Lundberg & Pollak, Duflo 2000, Doepke & Kindermann 2019)
4. Intimate Partner Violence (Card & Dahl 2011, Aizer 2010, Adams et al. 2024)
5. Gender Wage Gap (Blau & Kahn 2000, Niederle & Vesterlund 2011, Goldin 2014)
6. Motherhood Wage Penalty (Kleven et al. 2019, 2021, 2024)
7. Gender, Norms & Development (Jayachandran 2021, Bursztyn et al. 2020, Field et al. 2021)
8. SOGIE Economics (Badgett et al. 2024, Schilt & Wiswall 2008, Andresen & Nix 2022)
